Ti Ta Toggle (tt-Toggle)

Using awesome toggle buttons without Javascript or any other logic, other than plane form elements. The Pattern is the same as Twitter-Bootstrap. So it can be implementen without a breeze.

Default

The default Bootstrap way of generating checkbox looks a bit like this: 
			<div class="checkbox">
			<label>
			<input type="checkbox">default
			</label>
			</div>

Ti-Ta-Toggle awesomenes

Just by adding one <span> plus the class .checkbox-slider--default you get this result: 
			<div class="checkbox checkbox-slider--default checkbox-slider">
			<label>
			<input type="checkbox"><span>ttToggle example</span>
			</label>
			</div>

Indicator

Small visual que for the label to show that the checkbox is checked. Not so usable for the visually impaired, due to the fact it's only a color and contrast is to low.

usage

... <input type="checkbox" checked><span class="indicator-success">success</span> ...

CSS / LESS

To implement the ttToggle you can include the LESS file (FILE.LESS) into your project (that already uses Bootstrap). Or you can use the default CSS (FILE.CSS) output. https://github.com/kleinejan/titatoggle
